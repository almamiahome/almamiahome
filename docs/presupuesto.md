## Fundamentación del precio
- Cada módulo combina reglas de negocio + persistencia + validaciones + interfaz + documentación.
- El costo refleja complejidad de cálculo, impacto económico (premios/pagos) y criticidad operativa.
- En este dominio, un error de cálculo puede producir pérdidas económicas y pérdida de confianza, por eso se contempla QA fuerte.

## Retorno esperado
- Menos horas administrativas repetitivas.
- Menos errores en liquidaciones y reclamos.
- Mayor velocidad para cerrar campañas y pagar premios.
- Mayor capacidad de crecimiento sin aumentar proporcionalmente el trabajo manual.
